About SearchWord

SearchWord is a daily word search puzzle created for fun by a solo developer.

APIs & Services Used

This game uses the following public APIs and services to enhance your experience:

  • Free Dictionary API: Used to provide word definitions and phonetics.
    https://dictionaryapi.dev/
  • PurgoMalum Profanity Filter API: Used to filter out inappropriate words from the game.
    https://www.purgomalum.com/
  • Chart.js: For displaying the word length statistics chart.
  • Google Fonts: For the custom fonts used in the game.
  • Cookiebot: For managing cookie consent.
  • Google Analytics: For anonymous usage statistics.
  • Google AdSense: To help support the site.

This game is not affiliated with or endorsed by the above API providers.
